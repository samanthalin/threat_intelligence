Encyclopedia

Welcome to the Zyxel Encyclopedia.

In this encyclopedia, you can get the description of avirus and how ZyWALL defends against that particular virus.Search in the encyclopedia for virus/malware name or file hash of virus/malware.

What is virus/malware name?

For example, Wannacry or any partial name of a virus/malware.

How to search with file hash?

To search, just enter its hash. For example, ‘fe31f2fac5ca0919d29bcf1fe79b9645’

If you are a ZyWALL/USG or ATP users, and you would like to know the virus information detected by your device, you can search through encyclopedia.
